Coding examples:
Question: How do you define a function in Python?
Answer: In Python, you define a function using the def keyword followed by the function name and parameters. For example: def my_function(parameter): return parameter * 2
Question: What is a variable in programming?
Answer: A variable is a container that stores data values. In Python, you can create a variable by simply assigning a value: x = 10.
Question: How do you create a list in Python?
Answer: You create a list using square brackets. For example: my_list = [1, 2, 3]
Question: What is a for loop?
Answer: A for loop repeats a block of code for each item in a sequence. In Python: for item in my_list: print(item)

Problem solving:
Question: If I have 3 apples and give away 1, how many do I have?
Answer: If you have 3 apples and give away 1, you have 2 apples remaining.
Question: A train travels 120 miles in 2 hours. What is its speed?
Answer: Speed equals distance divided by time. 120 miles divided by 2 hours equals 60 miles per hour.
Question: What is 8 times 7 plus 10?
Answer: First multiply: 8 times 7 equals 56. Then add: 56 plus 10 equals 66.
